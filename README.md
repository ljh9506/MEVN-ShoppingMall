# Vue-ShoppingMall


```
Vue.js와 Express.js로 개발한 풀스택 쇼핑몰 사이트입니다.
JWT를 이용한 회원가입 로그인 기능, Vuex를 이용한 장바구니 기능, 이미지 업로드를 동반한 게시물 작성 및 리뷰 작성
기능 등을 구현하였습니다.
```

### 사용한 기술
```
- Vue.js
- Vuex
- Vuesax Css Framework
- Axios
- Express.js
- MongoDB
- AWS S3
- bcrypt
- multer
```
